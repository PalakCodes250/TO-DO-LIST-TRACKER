# TO-DO-LIST-TRACKER
The To-Do List Manager is a console app built using Python to assist in managing tasks throughout the process of adding, viewing, editing or completing them. It keeps track of due dates and priorities, and even reminds the user of overdue tasks. The application performs all of these tasks while saving all info so that managing tasks remains easy .
                  TO DO LIST TRACKER
Project Overview
 The To-Do List Manager is a Python console application that is meant to help users manage their daily work in an efficient way. Users can easily add tasks with all relevant information, including due date, category, and priority level. The app also allows them to edit or delete material and mark him or her tasks are completed. There are overdue reminders and overall summaries of total, pending, and completed tasks. Every task is saved in a file so that the application is straightforward, practical, and effective for basic task management.
FEATURES
Add New Tasks with information such as due date, category, and priority. See All Tasks in an organized list format. Update Existing Tasks to change details whenever needed. Remove Tasks that are no longer relevant. Check Tasks as Completed for reference later. Reminder List that shows any tasks due today or with a past due date. Task Summary that provides a total of tasks, completed tasks, and pending tasks. Data retained after program is closed so the tasks you have created are not lost. Functional Console Menu for simple navigation.
 
             Tools and Techniques Used
Menu-driven programming: This programming style offered the user the navigation to move back and forth between the different features.
 String Manipulation: String manipulation was a simple approach to keep the task information organized in a single string for the task details.
Input checking: Input checking was implemented for checking whether the user provided all the information required and if the data was in the accepted range.
 Modular Programming: Each of the three features, were coded as their module to keep a simpler clean module.
 Testing: Some tests were done manually to run the required feature, run per the length of data to check edge cases. mapping Test to output to reliability.
 
                         Steps to install and run the program
1. Download Python from python.org and add it to your PATH. 
2. Create a folder and create a file called todo.py. 3. Copy the program code into todo.py.
 4. Open your Terminal / CMD and navigate to the folder with the command: cd folder_path 
5. Run the program with this command: python todo.py
 
      Instructions For Testing
1. Execute the program by utilizing python todo.py. 
 2. Proceed to check all the menu options by working through each one (Add, View, Edit, Delete, Complete, Reminders, Summary).
 3. Add in a few example tasks with different dates, categories, and priorities.
 4. Edit a task and confirm that changes were properly reflected. 
5. Delete a task and confirm that it is no longer in the list.
 6. Mark a task as completed and confirm that the status is updated. 7. Check reminders to ensure that if you enter today's date that you see which task are due. 
8. Check task summary and confirm that the counts of total tasks, completed tasks, and pending tasks are correct. 
9. Save & Exit and re-run the program to ensure tasks have saved correctly.
